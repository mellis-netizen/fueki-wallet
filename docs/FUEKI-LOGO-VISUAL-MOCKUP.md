## Brand Consistency Check

### ✅ Logo Usage Guidelines

**DO:**
- Use official color codes (#2563EB, #3B82F6, #0F172A)
- Maintain 15% padding around icon edges
- Use provided SVG files for all scaling
- Keep octopus recognizable at all sizes
- Use high-contrast versions for small sizes

**DON'T:**
- Change colors or gradient direction
- Add extra elements (text, badges, effects)
- Distort aspect ratio (must be 1:1 square)
- Use low-resolution versions
- Add transparency to App Store icon
